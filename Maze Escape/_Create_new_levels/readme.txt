Here is how to create a new level:

1. Download the GPL editor Tiled: www.mapeditor.org

2. Open Level_base.tmx

3. Edit the level layers:
- Solide = the floor and walls 
- Fond = the background
- Objets = where the Start, Finish and elevators are

4. You can either replace an existing level or add the level and import it in XCode.

Optional: change the map dimensions, create your own sprites -- see the doc here: http://doc.mapeditor.org/
